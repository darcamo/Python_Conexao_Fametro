Imprimindo na tela

  • Em python imprimimos um valor usando a função print print("Tá muito fácil") Tá muito fácil
  • Múltiplos valores podem ser passados são separados por espaço x=10; print("Ele tinha", x, "anos") # Separador padrão: espaço Ele tinha 10 anos
  • Vejam mais opções da função print com print? no IPython

Lendo um valor do teclado

  • Em python recebemos uma entrada com a função input print("Digite o nome") nome = input() print("Digite a idade") x = int(input()) # Input retorna um texto print(nome, "tem", x, "anos") # Digitou 'John Doe' e '20' John Doe tem 20 anos
Nota
input sempre retorna uma string
