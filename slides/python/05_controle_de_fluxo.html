Controle de Fluxo em Python

Estruturas Condicionais

  • O escopo em Python é controlado por indentação
  • O caractere : inicia um novo escopo
  • Python possui as estruturas condicionais if, else e elif
    • elif corresponde a um else seguido de um if e existe para evitar indentação excessiva

Estruturas de Repetição

  • Python possui as estruturas de repetição usuais while e for
  • Em um bloco de código (corpo de um if, for, while, etc) podemos usar pass para não fazer nada útil para manter a sintaxe correta 
    if True:
    pass

Estruturas de Repetição

  • O for em Python itera através dos itens de uma sequência (uma lista, uma string, etc) na ordem em que eles aparecem
  • Para iterar em uma sequência de números use range
    for i in range(5, 8): ...: print(i**2) 25 36 49
